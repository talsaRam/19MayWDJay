Namespaces

  1. XML Namespace is used to avoid element name conflict in XML document.
  2. This attribute name must be started with "xmlns".
  3. Let's see the XML namespace syntax: <element xmlns:name = "URL">
  4. Uniform Resource Identifier is used to identify the internet resource. It is a string of characters.
  5. The most common URI is URL (Uniform Resource Locator) which identifies an internet domain address.

    Div Span
    The <div> tag is a block level element. The <span> tag is an inline element.

    Html Images

  1. Images can improve the design on the web page.
  2. The <img> tag is empty, it contains attributes only, and does not have a closing tag.
  3. The <img> tag has two required attributes:
    1. src - Specifies the path to the image
    2. alt - Specifies an alternate text for the image
    3. syntax: <img src="url" alt="alternatetext" >

    HTML-Form

  1. An HTML form is used to collect user input. The user input is most often sent to a server for processing.
  2. Get:
    1. The length of a URL is limited (2048 characters)
    2. Useful for form submissions where a user wants to bookmark the result
    3. GET is good for non-secure data, like query strings in Google
    1. Post:
      1. the submitted form data is not shown in the URL
      2. POST has no size limitations, and can be used to send large amounts of data.
      3. Form submissions with POST cannot be bookmarked

    Html-Header

  1. Html Title tag:
    1. The title tag define in the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
    2. The title tag is required in html document
    3. The contents of a page title is very important for search engine optimization (SEO)!

    Html Base:

  1. The <base> tag specifies the base URL and/or target for all relative URLs in a document.
  2. There can only be one single <base> element in a document, and it must be inside the <head> element.
